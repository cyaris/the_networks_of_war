# The Networks of War Frontend

This is the new Svelte frontend for the project. It replaces the legacy Jekyll/D3 include with a routed Svelte app and a first usable war browser.

## Run

From `frontend/`:

```sh
npm install
npm run dev
```

The app is available at `/the_networks_of_war` in development. The root route also renders the browser for convenience.

## Data

The frontend consumes `src/lib/static/graphData.json`. Do not place Python export code in this directory; the JSON is exported from the backend as part of Step 3.

From `frontend/`, regenerate the snapshot from an already-built backend database with:

```sh
npm run data:build
```

That delegates to `../backend/src/pipeline.py --step 3`, which refreshes the Step 3 tables and writes the frontend payload generated by `backend/sql/step_3/07_create_frontend_graph_data.sql`.

- `final_wars`
- `d3_war_nodes`
- `d3_war_links`

The generated shape is intentionally compact for the first screen. `d3_war_json` remains the natural backend table for an API route that serves one graph payload by `war_num`.
